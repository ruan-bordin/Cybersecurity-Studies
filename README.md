# Cybersecurity-Studies
🎯 Overview
This repository holds my study notes, CTF writeups, and certification progress. It is a living record of my learning journey: what I studied, what I struggled with, and what I want to learn next. The goal is to show curiosity, consistency, and steady progress.Built during my cybersecurity studies as a student in Brazil.
📚 Topics studied
Networking fundamentals (TCP/IP, DNS, ports, protocols).
Linux administration and Bash scripting.
Web vulnerabilities (OWASP Top 10 basics).
Cryptography concepts (hashes, encryption, TLS/SSL basics).
Privilege escalation fundamentals.
Incident response and SOC basics.
🔧 CTF platforms
TryHackMe — guided rooms and beginner-friendly challenges.
Hack The Box — more advanced machines and exploit practice.
PicoCTF / OverThe Wire — logic, Linux, and binary challenges.
PortSwigger Web Security Academy — web vulnerability labs.
📚 Certification roadmap
CompTIA Security+ — foundation and broad terminology.
eJPT or PNPT — practical penetration testing certification.
OSCP — long-term goal after gaining solid hands-on experience.
✅ Writeup format templateEach writeup follows this structure:
# CTF / Room Name

## 🎯 Objective
What was the challenge or goal?

## 🔧 Tools used
List the tools, commands, and scripts used.

## 📚 Steps taken
Describe the enumeration, exploitation, and privilege escalation process.

## ✅ Lessons learned
What did I learn? What mistakes did I make?

## 🚀 Next steps
What should I practice next based on this challenge?

🎯 How this repo helps my learningWriting things down forces me to truly understand the material, not just copy commands. It also makes it easy to review later, spot weak areas, and show my progress to mentors, recruiters, or myself.
